# 📊 AI Data Visualization Assistant

## 🚀 Overview
This project is a Streamlit-based web application that allows users to upload CSV datasets and automatically generate interactive visualizations, statistical summaries, and insights. It helps simplify data analysis by making it visual and easy to understand.

---

## 🎯 Features
- Upload CSV files
- Automatic data visualization (charts & graphs)
- Summary statistics of dataset
- Interactive and user-friendly UI
- Fast Streamlit dashboard

---

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Altair
